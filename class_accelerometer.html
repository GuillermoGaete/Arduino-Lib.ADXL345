Detailed Description

Represents a ADXL345 bassed Accelerometer sensor. You must call begin before calling any other methods.

int8_t Accelerometer::begin ( uint8_t  i2cAddr)

Attenpts to connect to the device. You must call begin before calling any other method.

Parameters
i2cAddrTakes the I2C address of the Gyroscope as an argument. Use OSEPP_ACC_SW_ON if your address switch is in the ON position. Use OSEPP_ACC_SW_OFF if your address switch is in the OFF position.
Returns
Returns 0 if it successfully connects to the Gyroscope sensor. Returns any other non-zero number if it fails. Check ADXL345lib.h for the various ERROR definitions.
